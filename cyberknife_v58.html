●

LIVE DATA

Real-time engine gauges, sensor readouts and fault codes. Tap any gauge or strip value for a full diagnostic description and direct navigation to the relevant tool. Long-press any gauge for resize / pin options.

⚙

TOOLS

Power Probe/DMM for circuit testing, Module scanner (ABS/SRS/TCU/BCM/TPMS), and ECU schematic/injector reference library.

▤

BUS/ECU

OBD-II PID requests, live J1939/CAN/LIN traffic, ECU VE/IGN/AFR maps, correction tables. Use ⚙ DECODERS to define custom CAN message rules — now with a form-based builder (no JSON required).

△

SCOPE/CAM

Oscilloscope (3ch), Data Logger with CSV import/replay, Thermal camera heat map, Borescope cylinder inspection.

▶

HARDWARE DOT — SIM / LIVE / ERR

The coloured dot top-right shows connection state. SIM = simulation mode (fully offline). LIVE = connected to real hardware via WebSerial. ERR = hardware error.

Tap the dot to connect a Raspberry Pi Pico (RP2040) running PiCCANTE firmware over USB serial at 115200 baud. The Pico streams KEY:VAL frames and accepts TX:ID:DATA commands.

◆

AI ANOMALY DETECTION (BETA)

CyberKnife now runs a per-CAN-ID anomaly stub in the background. It tracks running mean + variance for each signal and flags 3σ deviations to the debug log. This is the foundation for the self-learning FedLiTeCAN engine — tap badge ×3 to see alerts.

△

COMMUNITY SHARING

In BUS/ECU → CAN/LIN → PROTOCOL ANALYZER, enable CAPTURE then click SUBMIT UNKNOWN on any unidentified frame. A fingerprint JSON is copied to your clipboard — paste it to the community repo so everyone benefits.

♫

NAVIGATION

Use the rotary knob or tap the 5 section buttons. Sub-tabs switch content. Tap badge ×3 for debug log.
